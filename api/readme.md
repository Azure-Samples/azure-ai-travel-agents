## MCP Host

This package represents the MCP (Model Context Protocol). It contains the following MCP Clients:
- **Model Inference Agent**: This agent runs an LLM using Ollama on Azure Container Apps' serverless GPU for AI-powered responses.
- **Echo MCP Agent**: This agent echoes back any received input and is used as an example.
- **Customer Query Agent**: This agent handles customer queries and provides relevant information.
- **Destination Recommendation Agent**: This agent recommends travel destinations based on user preferences.
- **Itinerary Planning Agent**: This agent creates detailed travel itineraries based on user preferences.
